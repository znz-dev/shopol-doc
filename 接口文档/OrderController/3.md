# 显示指定订单详情

## 接口地址
```
GET /order/{orderId}

http://localhost:8080/order/5
```

## 请求参数
|参数名|参数类型|是否必填|参数描述|
|-----|------|-------|-------|
|{orderId}|Integer|Y|订单ID|
